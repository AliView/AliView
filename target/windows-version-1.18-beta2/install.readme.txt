Install:

Download and run AliView-Setup.exe

- The setup program will default installation to directory "C:\Program Files (x86)\AliView", and also ask if you want to create file associations for
alignment type files.



----*** ALTERNATIVE ***----
In directory "without_installer_version" is the program provided either as an .exe-file or a jar-file:

AliView.exe or aliview.jar (just download and run)
